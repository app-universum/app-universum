## Flow

The flow object creates a consistent vertical spacing between components.

`.au-o-flow`

<div class="au-o-box au-o-flow au-d-component">
  <p class="au-u-h4">Component 1</p>
  <p class="au-u-h4">Component 2</p>
</div>
